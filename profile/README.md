## About

The **Fabric Payment Standards Foundation** defines and promotes open protocols for digital payments designed to operate across systems, jurisdictions, and connectivity conditions.

Our mission is to establish a **neutral, vendor-independent standard** enabling:

* Seamless interoperability between payment providers
* Cryptographically secure transaction models
* Offline-capable value transfer mechanisms
* Scalable and decentralized financial infrastructure

---

## Core Specification Families

* **Stablecoin Stack (SSF)**
  Standard for user-centric, frictionless Stablecoin-based payment processing.

* **Cash-Pack Format (CPP)**
  Portable, chainable, and verifiable digital cash representation

* **MPC Signer Service**
  Distributed signature coordination and validation
